En Java, quelles sont les caractéristiques d'une interface ?

plusieurs réponses possibles

Une interface peut contenir des méthodes abstraites.
Une interface peut contenir des méthodes statiques.
Une interface peut contenir des variables d'instance.
Une interface peut être implémentée par plusieurs classes.
Une interface peut être instanciée directement.
Une interface peut contenir des méthodes par défaut.
